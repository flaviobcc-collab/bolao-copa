# Bolão Copa Cartola Web v2

## Vercel
Framework: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install
Root Directory: bolao-copa-cartola-v2 (se estiver dentro de subpasta)

## Variáveis de ambiente
VITE_SUPABASE_URL=https://SEU-PROJETO.supabase.co
VITE_SUPABASE_ANON_KEY=sb_publishable_...

## Supabase
Rode o arquivo `schema_supabase_v2.sql` no SQL Editor.
Depois de criar seu usuário, altere seu perfil para `admin` em `participantes`.
