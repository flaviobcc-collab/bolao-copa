Regras de Pontuação

Faça seus palpites e marque pontos conforme a proximidade do resultado real dos jogos.

📋 Como funciona

Você informa seu palpite antes do jogo. Após o resultado, sua pontuação é calculada automaticamente.

🎯

5 pontos

Na mosca

Você acertou o resultado do jogo!

Exemplo 1
Brasil 2 x 1 Argentina

Palpite: 2 x 1

Resultado: 2 x 1

✅ 5 pontos
Exemplo 2
França 0 x 0 Alemanha

Palpite: 0 x 0

Resultado: 0 x 0

✅ 5 pontos
Você acertou exatamente o placar da partida, por isso marcou 5 pontos.
⭐

3 pontos

Gols do vencedor

Você acertou o placar do vencedor!

Exemplo 1
Brasil 3 x 0 Japão

Palpite: 3 x 1

Resultado: 3 x 0

✅ 3 pontos
Exemplo 2
Inglaterra 2 x 0 Senegal

Palpite: 2 x 1

Resultado: 2 x 0

✅ 3 pontos
Você conseguiu acertar a quantidade de gols do time vencedor, mesmo errando o outro placar.
🤝

3 pontos

Indicação de empate

Você indicou empate e o jogo terminou empatado!

Exemplo 1
Croácia 2 x 2 Marrocos

Palpite: 1 x 1

Resultado: 2 x 2

✅ 3 pontos
Exemplo 2
Portugal 1 x 1 Uruguai

Palpite: 0 x 0

Resultado: 1 x 1

✅ 3 pontos
Você indicou corretamente que o jogo terminaria empatado. Se acertar o placar exato, será 5 pontos.
🥈

2 pontos

Gols do perdedor

Você acertou os gols do perdedor!

Exemplo 1
Brasil 2 x 1 Japão

Palpite: 4 x 1

Resultado: 2 x 1

✅ 2 pontos
Exemplo 2
França 3 x 1 Polônia

Palpite: 3 x 1

Resultado: 3 x 1

✅ 5 pontos
Você não acertou o placar correto, mas acertou a quantidade de gols do time perdedor.
✅

1 ponto

Indicação do vencedor

Você indicou o vencedor!

Exemplo 1
Brasil 1 x 0 Japão

Palpite: 2 x 0

Resultado: 1 x 0

✅ 1 ponto
Exemplo 2
Argentina 2 x 1 Arábia Saudita

Palpite: 3 x 0

Resultado: 2 x 1

✅ 1 ponto
Você não acertou o placar correto, mas ao menos indicou qual time venceria.
❌

0 ponto

Resultado incorreto

Você errou o resultado base do jogo!

Exemplo 1
Brasil 1 x 2 Japão

Palpite: 2 x 1

Resultado: 1 x 2

❌ 0 ponto
Exemplo 2
Alemanha 2 x 1 Costa Rica

Palpite: 1 x 0

Resultado: 1 x 2

❌ 0 ponto
Você errou o vencedor ou indicou empate quando houve vencedor. Por isso, não marcou pontos.

⚖️ Critérios de desempate

  1. Maior pontuação total
  2. Maior número de acertos “Na mosca”
  3. Maior número de acertos de 3 pontos
  4. Maior número de acertos de 2 pontos
  5. Maior número de acertos de 1 ponto

🏆 Premiação

A premiação será distribuída conforme o regulamento do bolão e os critérios informados no termo de participação.
